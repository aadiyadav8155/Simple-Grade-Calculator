# Simple Grade Calculator in C

A console-based Grade Calculator written in C.

## Features

- Input Validation
- Grade Calculation
- Ternary Operator
- Switch Case
- If-Else
- Console Output

## Grade Criteria

| Score | Grade |
|-------|-------|
| 85-100 | A |
| 70-84 | B |
| 55-69 | C |
| 40-54 | D |
| 0-39 | F |

## Compile

```bash
gcc Simple_Grade_Calculator.c -o grade
```

## Run

```bash
./grade
```

## Sample Output

```
Enter your score: 90
Your grade is A. Excellent work!
Congratulations! You are eligible for the next level.
```
